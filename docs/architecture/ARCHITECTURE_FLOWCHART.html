⑥ SSE 事件流時序圖

sequenceDiagram participant UI as 前端 UI participant API as /api/council/:id/run participant ENG as council.ts participant SEAT as Seat (AgenticRuntime) participant LLM as LLM API participant TOOL as Tool Handler participant MOD as Moderator (Opus) participant DB as PostgreSQL UI->>API: POST (SSE 連線) API->>ENG: runCouncilSession() ENG-->>UI: {type:"session_start"} ENG-->>UI: {type:"round_start", round:1} Note over ENG,SEAT: 並行啟動所有 seats loop 每個 Seat ENG->>SEAT: runSeatTurn(round=1) ENG-->>UI: {type:"turn_start", role, model} SEAT->>LLM: buildMessages() + stream LLM-->>UI: {type:"turn_delta", delta} (串流) opt 工具呼叫 SEAT-->>UI: {type:"tool_call", tool, args} SEAT->>TOOL: 執行工具 TOOL->>DB: rag_query / web_search DB-->>TOOL: 結果 TOOL-->>SEAT: 結果 + sourceRefs SEAT-->>UI: {type:"tool_result", result} Note over SEAT: 繼續 agentic loop end SEAT->>DB: saveTurn() SEAT-->>UI: {type:"turn_done", turn} end ENG->>ENG: classifyDivergence() ENG-->>UI: {type:"divergence_check", level, proceed_to_round2} alt divergence = moderate/high ENG-->>UI: {type:"round_start", round:2} Note over ENG,SEAT: 循序執行 seats（每個看到前面所有 R2） loop 每個 Seat (循序) ENG->>SEAT: runSeatTurn(round=2) Note right of SEAT: 同 Round 1 流程 end else divergence = low/none ENG-->>UI: {type:"round2_skipped"} end ENG-->>UI: {type:"moderator_start"} ENG->>MOD: buildModeratorPrompt() + stream MOD-->>UI: {type:"moderator_delta", delta} (JSON 串流) MOD->>DB: saveConclusion() ENG-->>UI: {type:"conclusion", conclusion} ENG-->>UI: {type:"session_done"}
